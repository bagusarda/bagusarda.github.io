Smart Home IoT: Voice Control & Energy Monitoring

Sistem otomatisasi rumah pintar berbasis Internet of Things (IoT) yang memungkinkan kontrol lampu via Google Voice Assistant. Selain kontrol suara, sistem ini dilengkapi fitur monitoring daya listrik *real-time* untuk memantau penggunaan arus, tegangan, dan daya (Watt) secara akurat menggunakan kombinasi sensor ZMPT101B dan ACS712.

Fitur Utama:

  • Kontrol lampu via perintah suara (IFTTT & Google Assistant)
  • Monitoring parameter listrik (V, I, P) di aplikasi mobile Blynk
  • Tampilan data lokal *real-time* via LCD 16x2
  • Kontrol manual *override* via aplikasi Blynk

Tech Stack:

  • Microcontroller: Wemos D1 Mini (ESP8266)
  • Sensors: ZMPT101B (Voltage) & ACS712 (Current)
  • Platforms: Blynk IoT, IFTTT, Google Home
  • Languages: C++ (Arduino IDE)

Alur Kerja & Hasil Analisis:

Sistem otomatisasi ini menggunakan platform IFTTT sebagai perantara web request (Webhooks API) ketika menerima trigger perintah suara dari Google Assistant, yang kemudian diteruskan langsung ke server Blynk Cloud untuk mengubah state relay pada pin D5 Wemos D1 Mini.

Untuk kalkulasi daya, data analog dari sensor arus (ACS712) dan sensor tegangan (ZMPT101B) diproses langsung oleh Wemos D1 Mini. Data hasil konversi tersebut kemudian dikalibrasi secara internal melalui kode pemrograman sebelum dikirimkan secara real-time ke dashboard Blynk IoT.

Soil Moisture Monitoring: Blynk IoT & Local Web Server

Sistem monitoring kelembapan tanah berbasis Internet of Things (IoT) yang dirancang untuk mendeteksi kadar air tanah secara real-time. Sistem ini mengintegrasikan mikrokontroler dengan sensor tanah untuk mendukung optimalisasi perawatan tanaman, di mana data hasil pengukuran dapat dipantau secara offline melalui LCD fisik maupun secara online melalui aplikasi Blynk dan halaman local web server.

Fitur Utama:

  • Monitoring nilai ADC, Tegangan, dan Status tanah secara real-time
  • Tampilan persentase kelembapan (0-100%) pada halaman web lokal
  • Antarmuka lokal menggunakan LCD 16x2 dengan modul I2C
  • Integrasi dual-sistem online via Blynk Cloud dan HTTP Web Server

Tech Stack:

  • Microcontroller: Wemos D1 Mini / NodeMCU (ESP8266)
  • Sensors: Soil Moisture Sensor (Analog Output A0)
  • Display: LCD 16x2 dengan Modul I2C
  • Platforms: Blynk IoT & HTTP Web Server Lokal
  • Languages: C++ (Arduino IDE)

Alur Kerja & Hasil Analisis:

Sistem bekerja dengan membaca sinyal analog dari sensor kelembapan tanah melalui pin A0 pada ESP8266 dengan range nilai ADC antara 0 hingga 1023. Data digital hasil konversi tersebut diproses langsung secara internal oleh mikrokontroler untuk dihitung ke dalam satuan tegangan nyata (0-5 Volt) serta skala persentase kelembapan tanah.

Kondisi tanah kemudian dikategorikan berdasarkan nilai pembacaan, di mana nilai ADC lebih besar atau sama dengan 510 akan menetapkan status Kering, sedangkan nilai kurang dari 510 memicu status Basah. Data status dan parameter numerik ini kemudian dikirim secara berkala ke server Blynk menggunakan fungsi `Blynk.virtualWrite()` serta dirender secara lokal ke halaman web server berbasis IP Address yang dapat diakses via browser.
